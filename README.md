# 🩺 DiaDoc

DiaDoc es una aplicación móvil diseñada para asistir a pacientes con diabetes en la gestión integral de su salud. La plataforma centraliza el monitoreo biométrico, la planificación nutricional y las rutinas de ejercicio, combinando herramientas de Inteligencia Artificial con funciones críticas de seguridad.

Este proyecto es desarrollado como parte de la cátedra Seminario Integrador de la carrera Ingeniería en Sistemas de Información (UTN - FRM).

## ✨ Características Principales

* **Autenticación y Perfiles Seguros:** Sistema de registro, inicio de sesión y gestión de perfiles médicos personalizados con restricciones y patologías (Módulo de Seguridad y Perfil de Usuario).
* **Motores de IA Generativa:** Generación automática de planes de alimentación y rutinas de ejercicio adaptadas a los datos biométricos del paciente.
* **Seguimiento Biométrico:** Bitácora para el registro diario de glucemia, comidas y actividades, con visualización en un Dashboard interactivo.
* **Módulo de Emergencias (S.O.S):** Emisión de alertas rápidas con geolocalización a contactos de emergencia preestablecidos.
* **Gestión de Catálogos:** Base de conocimientos administrable con escáner integrado para carga de información nutricional de alimentos.

## 🛠️ Stack Tecnológico

El proyecto está construido bajo una arquitectura moderna de desarrollo nativo para Android (MVVM):

* **Lenguaje:** Kotlin
* **Interfaz de Usuario (UI):** Jetpack Compose (Material Design 3)
* **Entorno de Desarrollo:** Android Studio
* **Backend y Base de Datos:** Firebase (Authentication & Cloud Firestore)
* **Control de Versiones:** Git / GitHub
